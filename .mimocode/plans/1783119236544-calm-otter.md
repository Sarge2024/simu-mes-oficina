# Plano: Corrigir favicon 404

## Problema
O navegador solicita `/favicon.ico` automaticamente (padrão HTTP), mas o projeto só possui `frontend/public/favicon.svg`. Isso gera erro 404.

## Solução
Criar um `favicon.ico` no diretório `frontend/public/` a partir do SVG existente.

### Passos
1. Converter `frontend/public/favicon.svg` para `favicon.ico` (32x32)
2. Manter o `<link rel="icon">` existente no `index.html` (funciona para navegadores modernos)
3. O `.ico` servirá como fallback para navegadores que solicitam `/favicon.ico`

### Arquivos
- **Criar**: `frontend/public/favicon.ico`

### Verificação
- Rodar `npm run build` e verificar que `favicon.ico` aparece em `dist/`
- Testar na Vercel: acessar `/favicon.ico` diretamente deve retornar 200
