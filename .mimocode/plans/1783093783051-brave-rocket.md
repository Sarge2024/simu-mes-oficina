# Plano: Modelo de Localização de Estoque

## Contexto

O controle de estoque atual é simplista: `Componente.estoque_atual` é apenas um contador inteiro. Não existe rastreamento de onde cada peça está fisicamente armazenada na oficina. O usuário solicita um modelo de codificação de localização com:

- **Fileira** (corredor/rua do depósito)
- **Lado** (direita/esquerda)
- **Nível** (andar da prateleira)
- **Bloco** (zona/seção do depósito)

## Design Decisões

1. **Modelo separado vs campos no Componente**: Um componente pode estar em múltiplas localizações (ex: 10 retentores na Fileira A, 5 na Fileira C). Portanto, `LocalizacaoEstoque` será um modelo separado com FK para `Componente`.

2. **Tenant-aware**: Sim — cada oficina tem seu próprio layout físico. Herda de `TenantModel`.

3. **Código de localização composto**: Os 4 campos (fileira, lado, nivel, bloco) formam um código legível como `A-ESQ-2-B1` (Fileira A, Esquerda, Nível 2, Bloco 1). Campo `codigo` calculado no `save()`.

4. **Componente continua com `estoque_atual`**: O contador total continua no `Componente` para performance. `LocalizacaoEstoque` opcionalmente terá `quantidade` para rastrear estoque por localização.

## Modelo `LocalizacaoEstoque`

```
Arquivo: suprimentos/models.py

LocalizacaoEstoque(TenantModel):
    componente    FK → Componente (CASCADE)
    fileira       Char(20)   — Ex: "A", "B", "AA"
    lado          Char(1)    — Choices: "D" (Direita), "E" (Esquerda)
    nivel         PositiveSmallIntegerField — Ex: 1, 2, 3
    bloco         Char(10)   — Ex: "B1", "B2"
    codigo        Char(50)   — Único por tenant, calculado: "{fileira}-{lado}-{nivel}-{bloco}"
    capacidade    PositiveIntegerField (default 0) — Capacidade máxima da posição
    quantidade    PositiveIntegerField (default 0) — Qtd atual nesta posição
    ativo         BooleanField (default True)
    criado_em     DateTimeField
    atualizado_em DateTimeField

    Meta:
        db_table = "sup_localizacao_estoque"
        constraints:
            UniqueConstraint(fields=["tenant", "codigo"], name="uq_sup_localizacao_tenant_codigo")
        ordering = ["fileira", "lado", "nivel", "bloco"]
```
